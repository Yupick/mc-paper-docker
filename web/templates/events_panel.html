Configuración de Eventos
Edita la configuración de eventos en formato JSON. Incluye fechas de inicio/fin, mobs especiales, drops exclusivos y recompensas.
Estructura de Evento:
{
  "id": "event_id",
  "name": "Nombre del Evento",
  "description": "Descripción",
  "startDate": "2025-12-20T00:00:00",
  "endDate": "2025-12-26T23:59:59",
  "enabled": true,
  "autoStart": true,
  "customMobs": [
    {
      "mobId": "mob_id",
      "spawnChance": 0.3,
      "spawnRadius": 50
    }
  ],
  "exclusiveDrops": [
    {
      "itemName": "Item Name",
      "dropChance": 0.5,
      "minAmount": 1,
      "maxAmount": 3
    }
  ],
  "rewards": {
    "eventCoinsPerKill": 5,
    "bonusXPMultiplier": 1.5,
    "bossEventCoins": 100
  }
}
